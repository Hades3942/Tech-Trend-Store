# Tech Trend Store

Tech Trend Store is a responsive web project built with **HTML, CSS, and JavaScript**.  
It showcases a modern online store layout with:

- A navigation bar with links to Home, About Us, Products, Reviews, and Contact.
- A hero section with a banner image, title, and "Shop Now" button.
- A product section with category filters (Laptops, Phones, Headphones, Desktops).
- A customer reviews section with profile images and testimonials.
- A footer with social media links and a newsletter subscription form.
- JavaScript interactivity including dark mode toggle, scroll-to-top button, form validation, and a welcome popup.
